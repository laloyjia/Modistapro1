Configurar Firebase para ModistaPro

Guía paso a paso — 15 minutos — Gratis

✅ Con Firebase tendrás: base de datos en la nube, login seguro con correo, acceso desde cualquier dispositivo, y respaldo automático de todos tus datos.
1
Crear cuenta en Firebase (es gratis)

Firebase es un servicio de Google. Si tienes cuenta de Gmail, ya estás casi listo.

🌐 Abrir Firebase Console

Haz clic en "Crear un proyecto" → ponle un nombre (ej: modistapro) → Continuar → puedes desactivar Google Analytics → Crear proyecto.

2
Registrar tu sitio web en el proyecto

Dentro del proyecto recién creado, busca el ícono </> (Web) en la pantalla principal y haz clic.

Ponle un apodo (ej: modistapro-web) → Registrar app.

Firebase te mostrará un bloque de código como este — copia esos valores:

// Esto aparecerá en tu pantalla con tus datos reales: const firebaseConfig = { apiKey: "AIzaSyXXXXXXXXXXXXXXXXXXX", authDomain: "miproyecto.firebaseapp.com", projectId: "miproyecto", storageBucket: "miproyecto.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abcdef" };
3
Pegar la configuración en ModistaPro

Abre el archivo ModistaPro/js/firebase-config.js con el Bloc de Notas (clic derecho → Abrir con → Bloc de notas).

Reemplaza los valores PEGA_AQUI_TU_... con los que copiaste de Firebase. Guarda el archivo.

⚠️ No compartas este archivo con nadie — contiene las claves de tu proyecto.
4
Activar el login con correo y contraseña

En Firebase Console, en el menú izquierdo haz clic en Authentication → pestaña Sign-in method.

Haz clic en Correo electrónico/contraseña → activa el primer interruptor → Guardar.

5
Crear la base de datos Firestore

En el menú izquierdo, haz clic en Firestore Database → Crear base de datos.

Elige "Comenzar en modo de producción" → selecciona la región southamerica-east1 (São Paulo, la más cercana) → Habilitar.

6
Configurar las reglas de seguridad de Firestore

En Firestore → pestaña Reglas. Borra lo que hay y pega esto:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }

Haz clic en Publicar. Esto garantiza que cada dueña solo ve sus propios datos.

7
Subir el sitio a internet con Netlify

El sitio ya está listo para publicarse. La forma más fácil:

  1. Ve a netlify.com → crear cuenta gratis (puedes usar Gmail)
  2. En el panel, arrastra la carpeta ModistaPro completa al área que dice "Arrastra tu sitio aquí"
  3. En ~30 segundos tendrás una URL pública tipo https://tu-negocio.netlify.app
  4. En Netlify → Site settings → puedes cambiar el nombre a algo más fácil
✅ Una vez publicado, agrega la URL de Netlify en Firebase Console → Authentication → Settings → Authorized domains → Add domain. Esto evita que otros sitios usen tu Firebase.
8
Crear la primera cuenta (la de la dueña)

Abre tu sitio publicado y ve a /login.html. Haz clic en la pestaña "Crear cuenta".

Ingresa el correo y contraseña de la dueña del negocio → Crear cuenta. Eso es todo — ya puede ingresar al panel y sus datos se guardarán en la nube.

⚠️ Anota bien el correo y contraseña. Si la olvidas, puedes hacer clic en "¿Olvidaste tu contraseña?" y Firebase enviará un correo de recuperación.
🎉
¡Listo! Tu ModistaPro ya está en la nube.

Plan gratuito de Firebase incluye: 50.000 lecturas/día, 20.000 escrituras/día, 1 GB de almacenamiento.
Más que suficiente para un negocio de confecciones.
